# 🐾 CAT-RECOVERY-SYSTEM 

Ультимативная система реанимации ноутов и ПК на базе собственного языка **CSP** (Cat Scripting Protocol). Защищено от "Конора с вилкой", скачков напряжения и кривых обновлений BIOS.

## ⚠️ ВНИМАНИЕ: ИНСТРУКЦИЯ ПО ЗАПИСИ (СТРОГО!)

Чтобы магия сработала и HEX-коды подтянулись корректно, записывать образ нужно **ТОЛЬКО** так:

1.  Используйте программу **UltraISO**.
2.  Откройте в ней файл `CatRecoveryPC.iso`.
3.  Перейдите в меню **Самозагрузка** -> **Записать образ Жесткого диска...**
4.  В поле **Метод записи** ОБЯЗАТЕЛЬНО выберите **USB-ZIP** (или USB-ZIP+). 
    *   *Никаких USB-HDD! Кот любит режим дискеты.*
5.  Нажмите **Записать**.

## 🛠 Технические характеристики:
*   **Язык:** CSP (Cat Scripting Protocol)
*   **Ядро:** INIT_UUT с динамической HEX-авторизацией.
*   **Контейнер:** .JCP (AES-256 encrypted).
*   **Вес:** Всего 66 КБ — легче, чем совесть твоего соседа.
